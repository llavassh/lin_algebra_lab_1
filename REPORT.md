# Отчёт по лабораторной работе №1

## Постановка задачи

Реализовать методы Гаусса (с выбором ведущего элемента и без) и LU-разложения для решения систем линейных уравнений. Сравнить методы по времени выполнения и точности на случайных матрицах и плохо обусловленной матрице Гильберта.

## Описание алгоритмов

**Метод Гаусса без выбора ведущего элемента:** прямой ход обнуляет элементы под диагональю, затем обратный ход находит решение.

**Метод Гаусса с выбором ведущего элемента:** на каждом шаге выбирается максимальный элемент в столбце для повышения устойчивости.

**LU-разложение:** матрица раскладывается на произведение нижней и верхней треугольных матриц, что выгодно при решении множества систем с одной матрицей.

## Выводы

1. **Сравнительная эффективность:** Для одной системы все методы работают за сопоставимое время.

2. **Выбор ведущего элемента:** Критически важен для плохо обусловленных матриц. При n=15 на матрице Гильберта метод без выбора даёт погрешность 100%, тогда как метод с выбором обеспечивает невязку на уровне машинной точности.

3. **Преимущество LU-разложения:** При решении 100 систем с одной матрицей LU-метод быстрее метода Гаусса в **53 раза** за счёт однократного выполнения разложения. В матрицах размера 1000 метод Гаусса становится более эффективным.
