# Лабораторная работа №1: Решение систем линейных алгебраических уравнений

## Краткое описание

В работе реализованы три метода решения СЛАУ:
- Метод Гаусса без выбора ведущего элемента
- Метод Гаусса с выбором ведущего элемента по столбцу
- LU-разложение

Проведены эксперименты по сравнению времени работы и точности методов на случайных и плохо обусловленных матрицах.

## Инструкция по сборке и запуску

### Требования
- Компилятор g++ с поддержкой C++17
- Make (mingw32-make для Windows)

### Сборка
mingw32-make

### Запуск
mingw32-make run

### Очистка
mingw32-make clean
